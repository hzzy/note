# rc文件
很多linux或unix在启动时 都需要.rc后缀的出世文件或配置文件
rc是runcomm的缩写 即 run command (运行命令)
rc是很多脚本文件的后缀

## 例如 
### .bashrc
linux的bash shell启动后运行的脚本
### .nacorc
用户在自己家目录创建.nanorc文件 nano启动前会去读取此配置
每行一句配置语句 配置语句以 set 或 unset 开头 
set mouse 激活鼠标
set autoindent 激活自动缩进
set smarthome 激活只能home键
